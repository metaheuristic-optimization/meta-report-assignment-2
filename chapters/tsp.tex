\section{TSP}

For the TSP experiment the following parameters were used

\begin{itemize}
  \item 5 executions
  \item 5 minute stopping condition
  \item Initial Algorithm - Nearest Neighbours or Random Tours
\end{itemize}

The following files were used 

\begin{itemize}
  \item inst-0.tsp
  \item inst-13.tsp
\end{itemize}
